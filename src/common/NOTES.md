## 功能特性

这个 feature 会自动安装和配置以下工具：

- **eza**: 现代化的 `ls` 命令替代品，具有更好的输出格式和颜色支持
- **fzf**: 强大的命令行模糊查找工具
- **zoxide**: 智能的 `cd` 命令，可以记住常用目录
- **mise**: 多语言工具版本管理器（替代 asdf）
- **starship**: 快速、可定制的跨平台 shell 提示符
- **zsh plugins** (仅在使用 Oh-My-Zsh 时):
  - `zsh-autosuggestions`: 根据历史记录自动提示命令
  - `zsh-syntax-highlighting`: 命令语法高亮

## QA

### 1. 403 rate limit

从 [Github Personal Access Token](https://github.com/settings/tokens/new?description=MISE_GITHUB_TOKEN) 创建一个 Token，不需要任何权限范围，使用 `MISE_GITHUB_TOKEN` 环境变量，以便工具可以访问 GitHub API

```json
"ghcr.io/aliuq/devcontainer-features/common:0": {
  "MISE_GITHUB_TOKEN": "<your_token>"
}
```

[参考来源](https://mise.jdx.dev/getting-started.html#github-api-rate-limiting)

### 2. Trust 目录

将工作区目录添加到环境变量 `MISE_TRUSTED_CONFIG_PATHS` 中，以便工具可以访问这些目录

```json
"containerEnv": {
  "MISE_TRUSTED_CONFIG_PATHS": "${containerWorkspaceFolder}"
}
```
